**IconSwapperGui** is a powerful **Windows desktop application** for changing **shortcut and folder icons** without navigating Windows‚Äô properties menus. It supports **.lnk shortcuts** and **Steam .url files**, making it ideal for customizing application and game icons on your Windows desktop.

A standout feature of IconSwapperGui is **icon version management**, allowing you to track, preview, and restore **previously used icons** for any shortcut ‚Äî making experimentation safe and fully reversible.

### üîÑ Icon Swapper

## üèÅ Getting Started

1. Download the ZIP file from the [**Releases**](https://github.com/aj-phillips/IconSwapperGui/releases) page  
2. Extract the archive using Windows Explorer, 7-Zip, or WinRAR  
3. Run `IconSwapperGui.exe`  
4. Select your **shortcut folder**  
5. Select your **icon folder**  
6. Start swapping and managing icons for your Windows and Steam shortcuts
